    Sensors

    Sensor config

    In this section the sensors can be configured and written to a config file. For the sensors there are two files, one for each I2C bus (Wire and Wire1). The values already store in the config file are prefilled in the text boxes.

    I2C bus 1 (wire) - Sensor 0 - 7 - %STATUS_SENSOR_CONFIG1_FILE%


    I2C bus 2 (wire1) - Sensor 0 - 7 - %STATUS_SENSOR_CONFIG2_FILE%
