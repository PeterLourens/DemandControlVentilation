Valve Control

Open Source demand controlled ventilation system

Store actual valve position(s) in JSON file

The application should not start before the actual positions are set in the web interface to prevent they are operate outside their range. The range is (fully closed) to 24 (fully open). At first startup, the position of all valves should be entered after calibration (see below). After storing the complete set, a single valve position can be stored in the status file.

Manually move valves

This section is to manually move the valves to a position without registering this change in de settings file. This can be used to calibrate the valve positions, e.g. set the valves to the fully closed position (0) and then store these positions in the valve position status file JSON. You can move multiple valves at the same time and empty values are ignored (i.e. move 0 positions forward)
